# Twin Stopwatch
공부 시간을 측정하기 위한 Twin Stopwatch입니다. 정지 버튼이 없으며, SWITCH를 통해 쉬는시간/공부시간을 전환할 수 있습니다.  
Google OAuth를 활용하여 사용자 로그인 및 시간을 저장할 수 있으며, 웹으로 어떤 플랫폼에서든 접근 가능합니다.
